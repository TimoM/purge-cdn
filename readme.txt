Adds button to Adminbar to easily purge the cache of one defined KeyCDN zone.



== Description ==

Sometimes you have to make a quick change to a post or a page and need to reset CDN cache immediately. 

= What it does? =
Deletes the cache of a predefined zone with the simple press of a button

= Features =
* delete cache fas and easy


= System Requirements =
* PHP >=5.3
* WordPress >=4.3
